# Summary
The memorable seven equality constants.

## Sum of squares
"""
    (a + b)² = a² + 2ab + b² 
""" \
```sce.sum_of_squares()```

## Subtract of squares
"""
    (a - b)² = a² - 2ab + b² 
""" \
```sce.subtract_of_squares()```

## Subtract two squares
"""
    a² - b² = (a - b)(a + b)
""" \
```sce.subtract_two_squares()```

## Sum of cubes
"""
    (a + b)³ = a³ + b³ + 3ab(a + b)
""" \
```sce.sum_of_cubes()```

## Subtract of cubes
"""
    (a - b)³ = a³ - b³ - 3ab(a - b)
""" \
```sce.subtract_of_cubes()```

## Cube two sums
"""
    a³ + b³ = (a + b)(a² - ab + b²)
""" \
```sce.cube_two_sums()```

## Cube two subtracts
"""
    a³ - b³ = (a - b)(a² + ab + b²)
""" \
```sce.sube_two_subtracts()```
